# Food Ordering System (Flask + MySQL)

A simple food ordering web application built using Python Flask and MySQL.

## Features

- Customer Registration
- Dynamic Food Menu
- Order Placement
- Payment Record
- Image-based Menu UI

## Technologies Used

- Python
- Flask
- MySQL
- HTML / CSS
- Jinja2

## Project Structure
app.py
templates/
static/
database/

## Setup Instructions

1. Clone the repository
  git clone https://github.com/karthikeyan0105/food-ordering-system.git
2. Install dependencies
3. Import database

Import `database/zomato.sql` into MySQL.

4. Run the application
5. Open browser
